# 技能名称：电梯清单导出工具 (export_elevator_data)

## 1. 核心定位
本技能是系统中唯一的电梯数据查询与导出入口。它通过 Python 脚本直接访问 MySQL 5.7 数据库，并将生成的报表自动上传至阿里云 OSS。

## 2. 交互逻辑 (非常重要)
- **直接输出 URL**：脚本执行成功后会输出一个以 `https://****.oss-cn-shanghai.aliyuncs.com/` 开头的 HTTPS 链接。
- **回复准则**：Agent 必须原封不动地将该链接提供给用户，作为最终的下载地址。
- **禁止反问**：严禁反问用户数据来源、访问方式或文件格式。所有的系统配置（OSS、数据库）均已在后台完成。

## 3. 参数提取规范
- **region (地区)**: 
  - 必须参数。对应数据库 `be.equlocarecodName` 字段。
  - 示例：输入“普陀区” -> 提取为“普陀”。
- **start_date & end_date (日期范围)**:
  - 格式：`YYYY-MM-DD`。
  - **当前年份锚点**：当前为 2026 年。
  - **模糊处理**：
    - “今年1月” -> `2026-01-01` 至 `2026-01-31`
    - “25年12月到26年1月” -> `2025-12-01` 至 `2026-01-31`

## 4. 数据安全与处理
- **防 SQL 注入**：已通过 Python 参数化查询实现。
- **存储机制**：文件在本地生成后立即上传 OSS，随后物理删除本地副本。
- **文件名规范**：`Elevator_Report_{地区}_{时间戳}.xlsx`。

## 5. 异常处理引导
- 若脚本返回“未查询到数据”，请告知用户：“抱歉，在指定时间段内该地区没有符合条件的电梯检测清单。”
- 若出现 `Access Denied` 或 `Connection Error`，请提示用户检查后台 `.env` 配置或数据库状态。
- 如果用户没有提供地区，请反问：“请问您需要导出哪个地区的清单？（例如：普陀）”。
- 如果查询结果为空，脚本会返回“未找到相关数据”，请直接将此信息转达给用户，不要试图编造数据。
- **文件分发**：目前文件保存在服务器本地。如果用户需要下载，请告知其文件路径（或后续集成 OSS 链接）。

## 6. 开发者备注
- 核心逻辑：Python 3.x
- 依赖：pandas, mysql-connector-python, python-dotenv
