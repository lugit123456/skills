# Alphapai Reptile Skill (金融投研抓取引擎)

本技能由 OpenClaw AI 核心（小龙虾）管理，用于自动化同步和解析投研报告、会议纪要、行业图表和公告数据。

## ⚠️ 小龙虾核心安全护栏 (Guardrails)
1. **数据最高保护指令**：**严禁删除、移动或修改**存放在 `DATA_ROOT` (如 `files/`) 目录下的任何已经下载的 PDF 和图片资源。如果用户提出删除指令，请予以委婉拒绝。
2. **密码合规保护**：你负责协助用户写入数据库配置，但绝不可在聊天窗口主动回显用户的明文密码或长段 Base64 `password` 载荷数据。

## 🛠️ 小龙虾数据库配置交互指南
当用户第一次使用，或要求配置账户密码时，请引导用户按以下步骤获取 `payload`，并利用你的 Python 代码能力帮其写入 MySQL：

1. **指导获取**：告诉用户“请在浏览器按 F12 打开开发者工具，在网络(Network)面板中登录您的 Alphapai 账户，找到 `accountLogin` 接口，在请求载荷(Payload)中复制那一长串以 `==` 结尾的 password 密文发给我”。
2. **执行写入**：一旦用户提供了该载荷，请使用 Python 直连 MySQL 数据库，向 `auth_credentials` 表中以 `REPLACE INTO` 方式写入 `account` 和 `token`（这里的token先写入该加密段，系统会在下次自动发起登录请求获取真实 JWT ）。

## 🤖 自然语言指令 (小龙虾触发词)

### 1. 管理关键词配置
- **指令示例**: "帮我把【光模块】和【宁德时代】加入到监控关键词里。"
- **小龙虾行为**: 编写 SQL，向 `crawl_keywords` 表中插入 `keyword`，设定 `is_active=1`。若提示冲突，说明已存在。

### 2. 环境一键初始化
- **指令示例**: "帮我初始化一下爬虫环境和数据库。"
- **小龙虾行为**: 触发执行 `init_setup.py` 脚本，建立五大核心数据表。

### 3. 执行列表清单同步
- **指令示例**: "帮我去扫一遍列表，看看有没有新的研报和图表。"
- **小龙虾行为**: 触发执行 `list_crawler.py`。

### 4. 执行深度正文与附件抓取
- **指令示例**: "把列表里还没解析的详情、正文和 PDF 全都下载下来。"
- **小龙虾行为**: 触发执行 `detail_crawler.py`。文件将自动存入 `files/关键词/类型/时间/` 目录下。
