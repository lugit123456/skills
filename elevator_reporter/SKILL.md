# 技能名称：电梯清单导出工具 (export_elevator_data)

## 概述
本技能是一个 Python 开发的数据库工具，专门用于从 MySQL 5.7 数据库中提取电梯检测报告（test_report）并结合基础电梯信息（base_elevator）导出为标准 Excel 文件。

## 参数指南
- **region (地区)**: 
  - 必须参数。
  - 映射数据库中的 `be.equlocarecodName` 字段。
  - 示例：用户说“普陀”或“普陀区”，请统一提取为“普陀”。
  
- **start_date & end_date (日期范围)**:
  - 格式：`YYYY-MM-DD`。
  - **默认逻辑**：如果用户只说“1月份”，请根据当前年份（2026年）自动补全为 `2026-01-01` 至 `2026-01-31`。
  - **跨年处理**：如果用户说“从去年12月到今年1月”，请准确计算对应日期。

## 执行流程
1. **参数提取**：LLM 识别用户的自然语言，将时间、地区转换为标准参数。
2. **脚本调用**：OpenClaw 调用 `main.py` 并传入 JSON 格式参数。
3. **数据库查询**：执行防注入 SQL，通过 `GROUP BY tr.eleNo` 确保每台电梯仅显示最新的一条记录。
4. **Excel 生成**：文件保存在技能目录下的 `exports/` 文件夹中。

## 常见问题与对话引导
- 如果用户没有提供地区，请反问：“请问您需要导出哪个地区的清单？（例如：普陀）”。
- 如果查询结果为空，脚本会返回“未找到相关数据”，请直接将此信息转达给用户，不要试图编造数据。
- **文件分发**：目前文件保存在服务器本地。如果用户需要下载，请告知其文件路径（或后续集成 OSS 链接）。
